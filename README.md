## Getting Started

Clone repository:
```git clone https://github.com/mick3211/next-disney-plus.git```

Apply for an API key at [TMDB](https://www.themoviedb.org/documentation/api)

Create a file inside the root directory named ``.env.local`` and put your TMDB API key inside it: ``NEXT_PUBLIC_API_KEY = YOURKEY``

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
